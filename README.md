# .FLS-language
FLS is a simple, interpreted programming language designed for experimentation and learning. It is not intended to compete with established languages; instead, it focuses on being functional and lightweight.

This language includes basic features such as variables, functions, arrays, objects, and file system access. It was originally created as part of a personal project for a game, but can serve as a foundation for building or testing simple scripting systems.

⚠️ Note: FLS is intended as a learning tool and proof-of-concept, not for production use.
